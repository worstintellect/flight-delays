# Flight Delays Fall 2018

## Описание проекта

Проект выполнен в рамках практической работы по дисциплине «Машинное обучение и анализ данных».

Цель проекта — построить модель машинного обучения для прогнозирования задержек авиарейсов более чем на 15 минут.

## Используемые библиотеки

- pandas
- numpy
- matplotlib
- seaborn
- scipy
- scikit-learn
- catboost

## Структура проекта

- загрузка данных;
- анализ данных (EDA);
- создание новых признаков;
- обучение модели Logistic Regression;
- обучение модели CatBoost;
- оценка качества модели по ROC AUC;
- формирование файла submission.csv.

## Запуск проекта

1. Открыть ноутбук в Kaggle.
2. Подключить соревнование Flight Delays Fall 2018.
3. Выполнить все ячейки последовательно.
4. Полученный файл submission.csv отправить на Kaggle.
